# Hospedaje Restaurante El Hombre

Sitio web moderno para Hospedaje Restaurante El Hombre en Puerto López, Manabí, Ecuador.

## Stack Tecnológico

- **Next.js 16** (App Router + Turbopack)
- **React 19**
- **TypeScript** (strict mode)
- **Tailwind CSS v4**
- **Shadcn UI**
- **Framer Motion**
- **next-themes** (dark/light mode)

## Características

- Diseño full bleed inmersivo sin contenedores rígidos
- Sistema de temas dinámico (oscuro/claro)
- Hero fullscreen con barra de reserva flotante
- Sección de habitaciones con overlaid image blocks
- Preview del restaurante con 3 platos principales
- Subpágina `/restaurante` con carta completa, tabs y carrito express
- Experiencias turísticas con image overlays
- Testimonios flotantes con comillas decorativas
- Navbar glassmorphism con detección de scroll
- Responsive mobile-first
- Animaciones suaves con Framer Motion

## Despliegue

El proyecto se despliega automáticamente en Vercel al hacer push a la rama `main`.

## Desarrollo

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
